# affiliateWebApp

Un programa de afiliados para brindar puntos y recompensas a tus clientes por las compras realizadas.

Node version: 14.20.1

### Notas:

Recuerda instalar los modulos:

```
npm install
```

Modo de desarrollo:

```
npm run dev
```

Mantener actualizado tailwind en desarollo:

```
npm run tailwind
```
